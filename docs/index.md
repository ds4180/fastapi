## 🛠️ 사용된 기술 스택
우리 프로젝트는 가장 현대적이고 강력한 도구들을 사용합니다.
- **Backend**: `FastAPI` (파이썬 기반의 빠르고 현대적인 웹 프레임워크)
- **Frontend**: `Svelte 5` (가장 직관적이고 가벼운 반응형 UI 라이브러리)
- **Database**: `SQLAlchemy` & `Alembic` (데이터 관리와 버전 제어)
- **Environment**: `Docker` (어디서나 동일하게 작동하는 환경)
