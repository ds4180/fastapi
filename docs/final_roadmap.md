# 게시판 완성으로 가는 마지막 여정: 최종 로드맵 가이드

게시판의 기본 기능을 모두 익혔다면, 이제 실제 사용자들이 편리하게 사용할 수 있는 **'진짜 서비스'**로 거듭날 세 가지 핵심 기능을 배울 차례입니다.

---

### 🔍 똑똑한 통합 검색 (진행 중)
- **설명**: 제목, 내용뿐만 아니라 '글쓴이'로도 글을 찾아내는 강력한 검색 엔진입니다.
- **핵심**: SQLAlchemy의 `outerjoin`과 `or_` 연산자를 활용한 통합 검색 쿼리를 구현합니다.

### ✍️ 마크다운(Markdown) 에디터 (다음 단계)
- **설명**: 코드를 넣거나 서식을 입힐 수 있는 개발자 친화적 입력 환경입니다.
- **핵심**: `marked`와 `dompurify`를 사용한 안전하고 세련된 렌더링 시스템을 구축합니다.

---

## 2. 사용자 관리와 권한 (안정성의 완성)

### 👥 나만의 공간, 프로필 페이지
- **설명**: "내가 지금까지 어떤 활동을 했지?"를 한눈에 보여줍니다.
- **핵심**: 내가 쓴 글 목록과 내가 '좋아요'를 누른 글들을 모아보는 '활동 데이터 수집' 로직을 다룹니다.

### 👑 서비스의 수호자, 관리자 페이지
- **설명**: 서비스 전체를 관리하는 마법사 권한입니다.
- **핵심**: 일반 사용자는 들어올 수 없는 비밀의 페이지(`admin`)를 만들고, 불량 게시물을 제어하는 방법을 익힙니다.

### 🛡️ 보안의 정석, X-Accel-Redirect
- **설명**: 아무나 파일을 보지 못하게 FastAPI가 허락하고, 배달은 Nginx가 하는 고급 기술입니다.
- **핵심**: 비밀글에 첨부된 이미지를 안전하면서도 빠르게 보여주는 실무의 '비기'를 배웁니다.

---

## 3. 더 넓은 세상으로: 추천 기능들 (Bonus)

- **🖼️ 이미지 첨부 (완료)**: 서버 사이드 썸네일 생성 및 `deleted_` 접두어를 활용한 안전한 격리 정책을 포함하여 완벽하게 구현되었습니다. [미디어 가이드 보기](./step1_media_guide.md)
- **🔔 실시간 알림**: 내 글에 답변이 달리면 즉시 알려주는 '푸시 알림'의 기초를 배울 예정입니다.
- **🌓 다크 모드**: 밤에도 눈이 아프지 않게 배경색을 바꾸는 센스 있는 UI 전환 기능을 소개합니다.
