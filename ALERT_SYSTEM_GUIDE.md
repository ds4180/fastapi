# 🔔 업무 지시(Alert) 및 기강 확립 시스템 최종 명세서

이 문서는 프로젝트 내 **실시간 업무 지시** 기능을 완벽하게 마무리하고, 시스템의 안정성과 미학적 완성도를 확보하기 위해 작성된 **최종 설명서**입니다.

---

## 🚀 1. 핵심 기능 마무리 (Finalized Features)

### ① 실시간성 및 영구 삭제
*   **실시간 갱신**: 30초마다 새로운 지시 사항이 있는지 백엔드에 접속합니다.
*   **영구 삭제 및 중복 차단**: '다시 보지 않기' 클릭 시 LocalStorage에 영구 저장됩니다. 또한, 단순히 닫거나 자동 소멸된 알림도 **세션 메모리(SessionSet)**에 기록되어, 30초 폴링 시 서버로부터 중복 데이터가 들어와도 다시 나타나지 않도록 완벽 차단합니다.

### ② 프리미엄 시각 효과 (Rich Aesthetics)
*   **Level 2 (Banner)**: 상/하단 배너에 **쉬머(Shimmer)** 효과를 적용했습니다. 빛이 왼쪽에서 오른쪽으로 슥 훑고 지나가 시선을 끕니다.
*   **Level 4 (Modal)**: 징벌적 모달에 **펄스(Pulse)** 애니메이션을 추가했습니다. 기강을 잡아야 할 때 화면이 미세하게 두근거려 긴장감을 줍니다.

### ③ 레이아웃 푸시 (Y-Axis Push)
*   배너가 나타날 때 화면 내용을 덮는(`fixed`) 것이 아니라, **레이아웃 본문을 부드럽게 밀어내어(`slide`)** 내용을 가리지 않도록 구현했습니다.

---

## 🛠️ 2. 데이터베이스 및 안정성 설계

### ① 유연한 DB 대응
*   `position` (상단/하단) 컬럼이 실제 DB에 반영되지 않았더라도, 코드 레벨에서 **'top'**을 기본값으로 강제 주입하여 시스템이 멈추지 않도록 설계했습니다.

### ② 경로 필터링 (Route Logic)
*   컴포마(`,`)로 구분된 여러 경로를 지원하며, `*` 또는 비워두면 전역 공지로 작동합니다.
*   알림 관리 페이지(`/alert-manage`)에서는 작업 편의를 위해 모든 알림이 보이지 않도록 예외 처리했습니다.

---

## 🎨 3. UI/UX 디자인 가이드

### ① 햄버거 메뉴 및 2열 그리드
*   **햄버거 메뉴**: 표준 Bootstrap 방식을 유지하여, 페이지 이동 시 자동으로 닫히는 가장 익숙하고 안정적인 동작을 보장합니다.
*   **마법의 2열 그리드**: 햄버거 메뉴를 눌렀을 때, 항목들이 길게 늘어지지 않고 **왼쪽/오른쪽 2열로 정갈하게 배치**되어 한눈에 들어옵니다.

### ② 알림 등급(Level) 체계
*   **Lv.1 (Toast)**: 둥근 팝업, **10초** 후 자동 소멸.
*   **Lv.2 (Banner)**: 가로 꽉 찬 배너, 직접 닫기 전까지 유지. (레이아웃 밀어내기)
*   **Lv.3 (Modal)**: 중앙 공지사항.
*   **Lv.4 (Strict)**: **5초 강제 대기** 기능 포함. 성급하게 확인 버튼을 누르면 대기 시간이 리셋됩니다. (기강 잡기용)

## 📋 4. 프로젝트 관리 원칙
*   **단순함이 최고다**: 복잡한 상태 관리 라이브러리 대신 Svelte 5의 기본 룬(`$state`, `$effect`)과 표준 `fetch`를 우선하며, Bootstrap의 기본 기능을 존중합니다.
*   **직관적인 코딩**: 어떤 개발자가 와도 3분 안에 구조를 파악할 수 있도록 코드를 간결하게 유지했습니다.
